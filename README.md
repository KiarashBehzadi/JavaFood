# 🍔 JavaFood - Food Ordering System

A Java-based restaurant management and food ordering application developed as part of Quera College (Advanced Java Course).

##  Features
- **User Management:** Three roles (Admin, Customer, Restaurant Owner)
- **Restaurant Operations:** Add/remove food items, manage inventory, track opening hours
- **Order Processing:** Real-time order placement with stock validation
- **Discount System:** Percentage/amount-based discount codes with expiration
- **Rating System:** Customers can rate restaurants (1-5 stars)
- **Data Persistence:** JSON file storage using Gson library

## 🏗️ Project Structure

src/
├── Models/ # Core entities (Order, Food, Restaurant, Discount)
├── Users/ # User roles (Admin, Customer, RestaurantOwner)
├── Exceptions/ # Custom exceptions
└── JavaFood/ # Main application logic

## 🚀 How to Run
1. Clone the repository
2. Ensure Java 8+ is installed
3. Add Gson dependency
4. Run `AdminPanel.java`

## 🛠️ Technologies
- Java 8+
- Gson for JSON processing
- Object-Oriented Programming (OOP)
- Collections Framework
- File I/O operations
