# 🍔 JavaFood 

A Java-based restaurant management and food ordering application developed as part of [Quera College-Advanced Java College]
see: https://quera.org/college/landpage/19983/advanced-java

## Features
- User roles: Admin, Customer, Restaurant Owner
- Order processing with stock validation
- Discount codes (percentage/amount)
- Restaurant rating system (1-5 stars)
- JSON data persistence with Gson

## Technologies
- Java 8+, Gson, Collections Framework
